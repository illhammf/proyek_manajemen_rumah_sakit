# Manual Book

## Nama Aplikasi
Sistem Manajemen Rumah Sakit Berbasis OOP.

## Tujuan Aplikasi
Aplikasi ini membantu pengelolaan data dokter, pasien, obat, pemeriksaan, dan rekam medis sederhana secara digital.

## Langkah Penggunaan

### 1. Menjalankan Aplikasi
Buka terminal pada folder proyek, lalu jalankan:

```bash
python main.py
```

### 2. Mengisi Data Contoh
Pada menu utama, pilih menu `6. Isi data contoh`. Sistem akan membuat data dokter, pasien, obat, dan satu pemeriksaan selesai.

### 3. Mengelola Dokter
Pilih menu `1. Kelola dokter`. Pengguna dapat menambah, melihat, mencari, mengubah, dan menghapus dokter.

### 4. Mengelola Pasien
Pilih menu `2. Kelola pasien`. Pengguna dapat menambah, melihat, mencari, mengubah, dan menghapus pasien.

### 5. Mengelola Obat
Pilih menu `3. Kelola obat`. Pengguna dapat menambah obat, melihat stok, mengubah data obat, menambah stok, dan menghapus obat.

### 6. Mengelola Pemeriksaan
Pilih menu `4. Kelola pemeriksaan`. Pengguna dapat membuat jadwal pemeriksaan, menambah obat, menyelesaikan pemeriksaan, dan membatalkan pemeriksaan.

### 7. Membuat Laporan
Pilih menu `5. Laporan`. Sistem menyediakan laporan ringkas, laporan stok obat rendah, dan ekspor laporan TXT.

## Catatan Input
Format tanggal wajib menggunakan `YYYY-MM-DD`, misalnya `2026-07-15`.
